# Student Profile App - DevOps Project 🚀

This project demonstrates a complete CI/CD pipeline using:

* Flask (Backend)
* Docker (Containerization)
* Jenkins (CI/CD Automation)
* GitHub Webhooks (Auto Trigger)
* ngrok (Public URL for Jenkins)

## Workflow

1. Code pushed to GitHub
2. GitHub webhook triggers Jenkins
3. Jenkins pipeline:

   * Clones repository
   * Builds Docker image
   * Runs container
4. Application deployed automatically

## Output

App runs on:
http://localhost:5002
